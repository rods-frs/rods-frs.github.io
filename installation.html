Installation

The installation of ArchLinux can be really dificult and even frustrating depending of your knowledge in the IT area, but don't worry, Here is a comfy video followed by the step-by-step guide of how to install it in your system with the time in minutes of where every step goes in the video.

  • 0:00 – Introduction
  • 1:00 – Boot into Arch ISO (UEFI mode)
  • 2:30 – Connect to WiFi using iwctl
  • 4:00 – Set system clock: timedatectl set-ntp true
  • 4:30 – Partition disks with fdisk or cfdisk
  • 8:00 – Format partitions: 
            mkfs.fat -F32 /dev/sdX1
        mkswap /dev/sdX2
        mkfs.ext4 /dev/sdX3
  • 9:13 – Install microcode: 
            pacman -S intel-ucode   # Intel
        pacman -S amd-ucode     # AMD
  • 10:00 – Mount partitions: 
            mount /dev/sdX3 /mnt
        mkdir /mnt/boot
        mount /dev/sdX1 /mnt/boot
        swapon /dev/sdX2
  • 11:30 – Install base system: 
    pacstrap /mnt base linux linux-firmware
  • 13:00 – Generate fstab: 
    genfstab -U /mnt >> /mnt/etc/fstab
  • 14:00 – Chroot: 
    arch-chroot /mnt
  • 15:00 – Set timezone: 
            ln -sf /usr/share/zoneinfo/Region/City /etc/localtime
        hwclock --systohc
  • 16:30 – Localization: 
            # Edit /etc/locale.gen
        locale-gen
        echo "LANG=en_US.UTF-8" > /etc/locale.conf
  • 18:00 – Network config: 
            echo "myhostname" > /etc/hostname
        # Edit /etc/hosts
  • 20:00 – Set root password: 
    passwd
  • 21:00 – Install GRUB: 
            pacman -S grub efibootmgr
        grub-install --target=x86_64-efi --efi-directory=/boot --bootloader-id=GRUB
        grub-mkconfig -o /boot/grub/grub.cfg
  • 22:00 – Reboot: 
            exit
        umount -R /mnt
        reboot
